SimCLR（Simple Framework for Contrastive Learning of Visual Representations）是一种基于对比学习的自监督学习方法，由 Google 团队在 2020 年提出。它通过最大化同一图像的不同增强版本（正样本对）之间的一致性，同时最小化与其他图像的增强版本（负样本）之间的相似性，来学习有效的视觉表征。
